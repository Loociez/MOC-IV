MOC: Shared Account Access

To play on the shared account without seeing the real password, install a small browser script:

  1. Install the Tampermonkey browser extension.
  2. Click here to install the script: consty-autologin.user.js (Tampermonkey should offer an install prompt automatically).
  3. Go to play.consty.com.
  4. You'll see an "Enter Access Code" prompt instead of the normal login screen. Enter the access code you were given — the game will log in automatically.

Don't have the access code? Ask whoever shared this page with you.
