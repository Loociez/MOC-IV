First, install the Tampermonkey extension for your Chromium-based browser (such as Chrome, Edge, Opera, or Firefox).

Once installed, click the provided “Install” link for the script. You will be taken to a confirmation page where you can review the script’s code before proceeding. After verifying, click Install to add it to Tampermonkey.

After installation, a Tampermonkey icon will appear in your browser toolbar. Click the icon and ensure that the script is enabled. Once enabled, simply refresh Ody for the script to take effect.

If you wish to disable the script at any time, you can either toggle it off within Tampermonkey or remove it entirely. After making changes, refresh Ody again to apply them.
